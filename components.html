Component Options

Component

Le composant est au coeur de l'expérience Ui Bibz. Chaque élément du framework hérite de l'élément composant . Ce dernier peut contenir 3 arguments:

  • content (variable ou block)
  • options
  • html_options

    Component.new content, options = {}, html_options = {}
    # or
    Component.new(options = {}, html_options = {}) do
      content
    end
Documentation

Render method

La méthode render permet de générer le code html du composant. 

    Component.new('My content', { state: :primary }, { class: 'my-class' }).render
    # or
    Component.new({ size: 3 }, { id: 'my-id' }) do
      My content
    end.render
Documentation

Size

Certain composant peuvent avoir l'option size dans leurs paramètres. L'argument size contient 3 valeurs distincts:

  • :xs
  • :md
  • :lg

    Component.new content, size: :xs
    Component.new content, size: :md
    Component.new content, size: :lg
Documentation | buttons , dropdowns , glyphs , navs , stars

State

Certain composant peuvent avoir l'option state dans leurs paramètres. Cette option permet de changer la couleur de l'élément comme précisé dans la documentation Boostrap . L'argument state contient 7 valeurs distincts:

  • :primary
  • :secondary
  • :danger
  • :success
  • :info
  • :warning

    Component.new content, state: :primary
    Component.new content, state: :secondary
    Component.new content, state: :danger
    Component.new content, state: :success
    Component.new content, state: :info
    Component.new content, state: :warning
Documentation | buttons , dropdowns , glyphs , navs , stars

Glyph

Certain composant peuvent avoir l'option glyph dans leurs paramètres. Cette option permet d'ajouter un glyph à son composant. Vous pouvez écrire la valeur de cet argument de plusieurs manières, pour celà, veuillez vous référer au componsant glyph . 

    Component.new content, glyph: 'star'
    Component.new content, glyph: { name: 'star', size: 2, type: 'fw' }
Documentation | buttons , dropdowns , glyphs , navs , stars , cards

Status

    Component.new content, status: :active
    Component.new content, status: :disable
Documentation

Html_options

Vous avez la possibilité d'ajouter toutes les propriétés html. 

    Component.new content, {}, { class: 'my-component', data: { target: 'first' }}
    Component.new({}, { id: 'my-id' }) do
      content
    end

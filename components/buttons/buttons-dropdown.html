Buttons dropdown

Le composant button_dropdown est une extension de l'élément component.
Tout comme ce dernier, le composant button_dropdown possède des options et des html_options présentes sous forme de hash.
Il hérite d'ailleurs des options présentes dans l'élement component.

Button dropdown

Le bouton dropdown accepte plusieurs type de block :

  • Link
  • Divider
  • Header
  • Html

Le composant link est une extension de l'élément component.

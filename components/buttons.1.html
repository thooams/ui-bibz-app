Buttons

Use Bootstrap’s custom button styles for actions in forms, dialogs, and more. Includes support for a handful of contextual variations, sizes, states, and more.

Le composant button est une extension de l'élément component.
Tout comme ce dernier, le composant button possède des options et des html_options présentes sous forme de hash.
Il hérite d'ailleurs des options présentes dans l'élement component.

Button options

Button

L'option state a pour agument les symboles suivants :

  • :primary
  • :secondary
  • :success
  • :warning
  • :danger
  • :link
    button 'Primary', state: :primary
    button 'Secondary', state: :secondary
    button 'Success', state: :success
    button 'Warning', state: :warning
    button 'Danger', state: :danger
    button 'Link', state: :link
Documentation

Outline

Le fond coloré des boutons peut être retiré grâce à l'option outline qui possède pour argument un booléan.

    button 'Primary', state: :primary, outline: true
    button 'Secondary', state: :secondary, outline: true
    button 'Success', state: :success, outline: true
    button 'Warning', state: :warning, outline: true
    button 'Danger', state: :danger, outline: true
Documentation

Size

La taille peut être être définit â travers l'option size qui possède les 2 arguments suivants:

  • :lg
  • :sm
    button 'Large button', state: :primary, size: :lg
    button 'Medium button', state: :primary
    button 'Small button', state: :primary, size: :sm
Documentation

Type

Les boutons peuvent être présentés sous forme de block grâce à l'option type qui a pour argument le symbole :block.

    button 'Block level button', state: :primary, type: :block
    button 'Block level button', state: :secondary, type: :block
Documentation

Status

Les boutons hérite de l'option status présente dans l'élément component Cette option permet de desactiver ou d'activer l'élément concernant à travers ces 2 arguments :

  • :active
  • :disabled
    button 'Primary', state: :primary, status: :active
    button 'Primary', state: :primary, status: :disabled
Documentation

Options and html options

Tout comme le component, l'élément bouton peut s'écrire de 2 différentes manières:

  • en ligne
  • par block
    button 'Primary', { state: :success }, { class: 'button' }
     
    button({ state: :success }, { class: 'button' }) do

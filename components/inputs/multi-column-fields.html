Simple form

You can use the multi_column input with gem simple form.

You must add this line in the file "config/initializers/simple_form.rb"

  • Ruby
    ...
    config.custom_inputs_namespaces << "CustomInputs"
    ...

Then in your view, you can insert your input multi_column field.

  • Ruby
